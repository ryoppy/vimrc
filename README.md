vimrc
======================================

INSTALL
----

Use curl (for Mac OS X):

     curl -o - https://raw.github.com/ryoppy/vimrc/master/auto-install.sh | sh

or wget (for most UNIX platforms):

     wget -O - https://raw.github.com/ryoppy/vimrc/master/auto-install.sh | sh
